# Code of Conduct

Slavey is a technical project. Discussions should stay respectful, specific, and focused on improving the software.

## Expected Behavior

- Be direct without being hostile.
- Assume good intent until behavior proves otherwise.
- Critique code, designs, docs, and tradeoffs rather than people.
- Keep issue and pull request discussions on topic.
- Respect maintainers' decisions about scope, priorities, and project direction.

## Unacceptable Behavior

- Harassment, threats, stalking, or sustained disruption.
- Discriminatory comments or personal attacks.
- Publishing private information without consent.
- Sexualized content or attention in project spaces.
- Spam, trolling, or bad-faith participation.

## Enforcement

Maintainers may edit, hide, lock, or delete comments; close issues or pull requests; temporarily restrict participation; or permanently ban participants from project spaces.

If you need to report conduct concerns, contact a maintainer privately through GitHub. Do not include sensitive personal information in a public issue.

Maintainers should apply this policy consistently and proportionally, with the goal of keeping the project useful, safe, and technically productive.
